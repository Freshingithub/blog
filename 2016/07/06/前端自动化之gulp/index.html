前端自动化之gulp

发表于 2016-07-06 19:38

gulp能为我们做的工作

  • 校验 HTML、CSS、JavaScript 和 JSON 文件的语法。
  • 将 ES6 JavaScript 代码编译为 ES5（通过使用 Babel、Traceur 或者 TypeScript）。
  • 运行单元测试和端到端测试。
  • 合并和压缩 CSS、JavaScript 文件。
  • 通过 HTTP 来提供静态文件服务。
  • 执行 shell 命令。
  • 监听特定文件（或特定文件类型）的变动，当发现文件改变时运行特定任务。
  • 当这些有变化的文件被处理后刷新浏览器（LiveReload）。

安装gulp

  • CLI运行npm install -g gulp安装gulp。
  • npm install --save-dev gulp-[name]安装所需的gulp插件,--save-dev命令会在package.json文件中添加开发依赖。
