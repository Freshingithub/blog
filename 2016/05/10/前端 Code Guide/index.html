前端 Code Guide--

发表于 2016-05-10 19:14

HTML

语法

  • 把tab键设置为两个空格。

  • 嵌套元素应该缩进两个空格。

  • 属性值要用双引号括起来。

  • class命名只能出现小写和破折号(不能是下划线，驼峰命名法)

字符编码

通常是用UTF-8编码。UTF大写。

引入css和javascript

根据HTML5规范，不用再设置它们的type属性，因为当引入它们的时候已经有了默认的type属性。

属性顺序

  1. class

  2. id, name

  3. data-*

  4. src, for, type, href, value

  5. title, alt

  6. role, aria-*

布尔属性

XHTML要求给布尔属性添加值，但是HTML5没有这样的要求。简而言之，就是不要添加属性值。

减少标签和js生成标签

尽量使用最少的标签保持最小的复杂度，尽量避免用js生成标签。

CSS

语法

  • 同样使用两个空格作为tab。

  • 组织选择器时，每个选择器单独占一行。

  • 每一行只声明一条属性。

  • 逗号分隔的属性值之后应该有一个空格，而括号内的逗号之后不要留空格。

  • 小于1的小数前面不要加上0，用.5em代替0.5em。

  • 16进制颜色值用小写，用#fff代替#ffffff。

  • 属性值为0时不要加上单位，用0代替0px。

  • 选择器中的属性值用双引号包起来，如input[type="text"]。

属性的声明顺序

相关属性声明应该写在一起。

声明顺序如下:

  1. 位置声明(position)

  2. 盒模型声明

  3. 文字排版声明

  4. 视觉声明

不要使用@import

media query最好放置在相关规则旁边

使用简写属性时可能会覆盖原有的css样式，常见的属性有:

  • padding

  • margin

  • font

  • background

  • border

  • border-radius
