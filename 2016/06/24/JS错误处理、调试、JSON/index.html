JSON

JSON是一种数据格式，很多语言都有针对JSON的解析器和序列化器

它可以表示以下三种类型的值：

  • 简单值

JSON中必须使用双引号(单引号导致语法错误)

  • 对象

对象名必须加双引号，对象值可以为简单值，也可以为嵌入的对象

  • 数组
