apache web server

安装完成后，打开Apache安装文件夹进入conf文件夹找到httpd.conf文件，用文本编辑器打开，添加以下代码：

listen 80端口（IE专用）

Apache的web根目录是”/Apache/htocs/“ 如果需要修改，打开httpd.conf文件，修改DocumentRoot “/Apache/htocs/“ 为DocumentRoot “你的目录” 例如：DocumentRoot “/var/www/html”

mysql

官网提供Windows环境的安装包格式有两种：msi（图形安装界面）zip（直接解压就是安装好的，但是需要配置参数）

解压后复制my-default.ini到当前目录，并改名为my.ini 修改

以管理员身份打开CMD，将目录切换到D:\Mysql\bin下，运行mysqld -install安装sql服务（mysqld -remove删除服务），运行net start mysql是启动服务（stop是停止）。

重点：：：：：：如果启动服务时出现NET HELPMSG 3534错误，则采取以下方法：

首先运行mysqld -remove删除服务，然后运行mysqld –initialize 进行初始化（注意是两个“-”符号），然后就会发现目录下多了一个data文件夹，里面有很多文件。再安装启动服务。
