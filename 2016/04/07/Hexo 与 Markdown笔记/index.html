Hexo 与 Markdown笔记---

发表于 2016-04-07 22:55

Markdown基础语法

  1. 分段：两个回车
  2. 换行：另起一行 + 两个空格 + 回车
  3. 标题：#~######井号的个数表示几级标题
  4. 引用：>
  5. 列表：有序1.，无序-，+，*，后面有个空格
  6. 代码区块：四个空格开头
  7. 链接： [文字](链接地址)
  8. 图片：![图片说明](图片地址)，图片地址可以是本地路径也可以是网络地址
  9. 强调：**文字**(粗体)，_文字_，*文字*（斜体）, ***文字***(粗体+斜体)
  10. 代码：`包围代码
  11. 脚注：hello[^hello] + 回车 + [^hello]: h1

Markdown文件在线编辑器：传送门

Hexo常用命令

1、新建

hexo new blogname
新建的文件在hexo/source/_posts/下

2、编译

hexo generate
编译生成的html文件等在public目录下

3、开启本地服务

hexo server
默认是4000端口，但是我在部署时遇到端口被占用的情况，后面在知乎上找到解决方法
修改hexo/node_modules/hexo-server/index.js中的port项即可

4、部署

hexo deploy
首先配置好git，我直接copy public文件夹里面的内容到.deploy_git文件夹下，然后add commit push
